# miApp-evaluacion-cloud

Aplicación full-stack con Docker Compose, PHP (API REST) y React.

## 🚀 Tecnologías

- PHP 8.2 + Apache
- MariaDB 11
- React 18
- Docker Compose

## 📁 Estructura

- `backend/`: API REST en PHP
- `frontend/`: Aplicación React
- `bash/`: Scripts de automatización
- `docker-compose.yml`: Orquestación de contenedores

## ⚙️ Requisitos

- Docker y Docker Compose instalados
